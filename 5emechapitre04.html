Chapitre : Addition de nombres relatifs

À retenir

1er cas : même signe

  • Je garde le signe commun
  • J’additionne les valeurs

Exemples :

  • (+5) + (+9) = +14 => Je gagne 5 + Je gagne 9 donc je gagne 14
  • (-3) + (-4) = -7 => Je perds 3 + Je perds 4 donc je perds 7

2ème cas : signes contraires

  • Je garde le signe du nombre le plus grand en valeur
  • Je soustrais les valeurs

Exemples :

  • (+4) + (-7) = -3 => Je gagne 4 + Je perds 7 donc je perds 3
  • (-5,6) + (+3,2) = -2,4 => Je perds 5,6 + Je gagne 3,2 donc je perds -2,4

Cas particulier :

Deux nombres opposés ont pour somme 0.

  • (-4) + (+4) = 0 => Donc -4 et +4 sont deux nombres opposés

Astuce :

On peut changer l’ordre des nombres dans une addition. Et donc calculer ensemble tous les positifs d'un côté et ensemble tous les négatifs de l'autre côté

  • (+4) + (-7) + (-3) + (+15) + (+1) + (-11)
  • (+4) + (+15) + (+1) + (-7) + (-3) + (-11)
  • (+20) + (-21)
  • -1
