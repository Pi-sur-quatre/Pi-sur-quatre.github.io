Soustraire deux nombres relatifs

À retenir

Soustraire un nombre relatif revient à additionner son opposé

Exemples :

  • A = (-2) - (-3) -> On veut soustraire le nombre -3
  • A = (-2) + (+3) -> On additionne l'opposé de -3 donc +3
  • A = 1 -> On calcule

2ème cas : signes contraires

  • Je garde le signe "du plus fort"
  • Je soustrais les valeurs

Exemples :

  • (+4) + (-7) = -3 => Je gagne 4 + Je perds 7 donc je perds 3
  • (-5,6) + (+3,2) = -2,4 => Je perds 5,6 + Je gagne 3,2 donc je perds 2,4

Cas particulier :

Deux nombres opposés ont pour somme 0.

  • (-4) + (+4) = 0 => Donc -4 et +4 sont deux nombres opposés

Astuce :

On peut changer l’ordre des nombres dans une addition.
Et donc calculer ensemble tous les positifs d'un côté et ensemble tous les négatifs de l'autre côté

  • (+4) + (-7) + (-3) + (+15) + (+1) + (-11)
  • (+4) + (+15) + (+1) + (-7) + (-3) + (-11)
  • (+20) + (-21)
  • -1
